# Windows关闭自动更新
## 方法一：禁用Windows 更新服务
按下 Win + R 键，在运行框中输入 services.msc，然后按Enter 键打开“服务”窗口。 
在服务列表中找到“Windows Update”。
双击该服务，在“启动类型”下拉菜单中选择“禁用”。 
如果服务正在运行，请点击“停止”按钮。 
切换到“恢复”选项卡，将“第一次失败”、“第二次失败”和“后续失败”全部设置为“无操作”。 
点击“应用”和“确定”保存更改。 

## 方法二：使用本地组策略编辑器(适用于专业版及更高版本)
按下 Win + R 键，在运行框中输入 gpedit.msc，然后按Enter 键打开本地组策略编辑器。 
导航至 计算机配置 > 管理模板 > Windows 组件 > Windows 更新。 
找到“配置自动更新”并双击打开。 
选择“已禁用”，然后点击“应用”和“确定”。 
（可选）您还可以找到“删除使用所有Windows更新功能的访问权限”并启用它，以进一步限制更新的访问。 

## 方法三：禁用任务计划程序中的更新任务
按下 Win + R 键，在运行框中输入 taskschd.msc，然后按Enter 键打开任务计划程序。 
导航到 任务计划程序库 > Microsoft > Windows > Windows Update。 
在右侧窗口中，右键单击“Schedule Start”任务，然后选择“禁用”。
